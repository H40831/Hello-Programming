プログラミング言語って？

プログラミングは、文字のデータ（＝テキストファイル）を使って
プログラムに実行させたい命令を書いていく作業になります。
実行させる命令は、日本語じゃなく、これから学習する書き方で書いていくことになりますが
この書き方のことを、プログラミング言語と呼びます。
今後「言語」という言葉がでてきたら、プログラミング言語のことを言っていると考えてください。

これから学ぶ言語

プログラミング言語は、やりたいことによって使い分ける必要があります。
このカリキュラムでは、WEBサイトやWEBアプリを作るための。
HTML、CSS、JavaScriptという、3つの言語を勉強していきます。

HTML

HTMLは、WEBサイトの構造を決めるための言語です。
文字、ボタン、入力欄などの要素を配置する順番を書きこんだり、
このスライドの水色の部分のように、要素の範囲を決めたりします。
更に、WEBサイトの設定（タイトルの指定や、CSSやJavaScriptの読み込みなど）も、HTMLで行っています。

イメージ

CSS

CSSは、WEBサイトの見た目やデザインを決めるための言語です。
背景や文字の色、要素の右寄せや中央寄せ、要素の間隔･余白や表示/非表示の指定をはじめ、全てのデザインはCSSで指定していきます。
また、マウスが合っている要素の色を変えたり、アクセス済のリンクの色を変えるなど、ユーザーの動きに対して見た目を変えるような指定もできます。

イメージ

JavaScript

JavaScriptは、HTMLやCSSでは出来ない様々な機能を作る言語です。
ページの移動、テキストのコピー、要素の追加、数をカウントして表示するなどの様々な機能を作成したり、その機能を発動する条件やタイミングを指定するなど…。１つ１つの機能を組み合わせれば、できることは無限にあります。
（JavaScriptを簡単に使用するためのjQueryという機能も学びます。）

この３つは
プログラミング言語か？

実は、この３つの中でプログラミング言語と呼ぶものはJavaScriptだけです。
一般的にプログラミングというのは「機能」を作ることを指し、
HTMLやCSSで出来ることは、プログラミングとは呼びません。
HTMLはマークアップ言語、CSSはスタイルシート言語と呼ばれています。

３つの言語で作れるものと、
作れないもの。

この３つの言語が使えるようになると、WEBサイトやWEBアプリで表示されているWEBページが作れるようになります。
まずは、最初のゴールとして意識してみてください。
サーバーが必要な機能を作りたい場合は、
この後にサーバーサイドのプログラミング言語(PHP、Ruby、Pythonなど)を学んでいきましょう。

サーバーとは (1)

WEBでは、どんな流れでパソコンにWEBサイトを表示しているでしょうか？
私たちがWEBサイトを表示するとき
1. URLにアクセスする。
2. WEBページを表示する。
という流れでパソコンにWEBサイトを表示していますが、
これを更に細かい流れに分けると、下図のようになります。

あなたのPC
(ローカルやクライアントなどと呼ぶ)

URLを入力したとき、同時に
WEBサイトのデータを返送するよう
サーバーにリクエストを送ります。

「http://◯◯◯.com/index」というアドレスの場合、
「http://」が、リクエストしているという合図、
「◯◯◯.com」が、リクエストするサーバーのアドレス、
「index」が、返送してほしいファイルの名前
を表しています。

サイト運営会社などが所有するPC
(サーバーと呼ぶ)

サーバーは、リクエストを受け取り、
あなたに送るべきデータを選んで
返送します。

サーバーが送らなかったファイルは
あなたのPCから表示することはできないので、
サーバーは個人情報やパスワード、
見せたくないファイルなどを守ることができます。

WEBサイトのデータ

WEBサイトのデータが、
あなたのPCに送信されます。

WEBサイトのデータというのは、
3つの言語が書かれたテキストファイルや、
サイトに表示する画像ファイルなどの事です。

あなたのPC

あなたのPCが、自動で
ブラウザ上にデータを開きます。

データを開くというのはつまり、
HTMLで決められた構造、CSSで指示されたスタイル、
JavaScriptで作られた機能を、
言語の指示通りに表示するという事です。

サーバーとは (2)

スマホにアプリをインストールするのと同じように、
PCに、サーバーとして利用するための機能をインストールすることで、
そのPCはサーバーになります。
また、その他にも様々な機能を導入することができます。
サーバーの代表的な役割が以下の３つになります。

サーバーの主な役割

スマホにアプリをインストールするのと同じように
PCにサーバーの機能を導入することで、
リクエストしてきたユーザーのPCに対して
任意のファイルを送ることが出来ます。
個人情報やパスワードなど、機能として必要だけど
他の人に見せたくないデータは、送らないようにしておくことで
守ることができます。

データの共有

ユーザーが入力･送信したデータをサーバーに保存し
保存したデータを他のユーザーに送ることで、
サーバーにアクセスしてきた沢山のユーザーと
データを共有することができます。
オンラインゲームや、掲示板、チャットなどで
使われている仕組みです。

データベース

ユーザーひとりひとりの情報を扱う場合、
とても膨大な数の情報を扱うことになりますが、
その情報を取り出すのに時間をかけるわけにはいきません。
膨大な情報から、特定の条件に合致する情報を素早く取り出すために
データーベースという機能があります。
通常、ユーザーの個人情報や、パスワードなどの情報は、
このデータベースに格納されている事がほとんどです。
