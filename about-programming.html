プログラミング言語って？

プログラミングは、文字のデータ（＝テキストファイル）を使って
プログラムに実行させたい命令を書いていく作業になります。
実行させる命令は、日本語じゃなく、これから学習する書き方で書いていくことになりますが
この書き方のことを、プログラミング言語と呼びます。
今後「言語」という言葉がでてきたら、プログラミング言語のことを言っていると考えてください。

これから学ぶ言語

プログラミング言語は、やりたいことによって使い分ける必要があります。
このカリキュラムでは、WEBサイトやWEBアプリを作るための。
HTML、CSS、JavaScriptという、3つの言語を勉強していきます。

HTML

HTMLは、WEBサイトの構造を決めるための言語です。
文字、ボタン、入力欄などの要素の配置する順番を書きこんだり、
要素のまとまり（このスライドでいうと水色の部分など）の範囲を決めたりしています。
HTMLでは、WEBサイトのデザインを指定しないため、ここで配置された要素は、あらかじめ決められたデザイン通りに表示されるだけになります。
もうひとつの役割として、WEBサイトの設定（タイトルや、読み込むCSSファイルやJavaScriptファイルなど）をする部分も、HTMLで行っています。

CSS

CSSは、WEBサイトのデザインを決めるための言語です。
HTMLで決めた構造に対して、どういうデザインを指定するかを決めます。
背景色や文字の色、背景画像、要素の右寄せや中央寄せ、要素の間隔（余白）の指定などをはじめ、ほぼ全てのデザインをCSSで指定していきます。

また最近では、マウスが合っている要素だけ色を変えるとか、
一度アクセスしたことのあるリンクだけ文字色を変えるなど、
ユーザーの動きに対してデザインを変えるような指定もできるようになってきました。

JavaScript

JavaScriptは、HTMLやCSSでは出来ない、いろんな機能をつくるためのプログラミング言語です。
（ボタンをクリックしたら）テキストをコピーする。
要素を後から追加する、要素の数をカウントして表示する。
スライドしたら次のページを表示する。
など、とにかく色んなことができます。
